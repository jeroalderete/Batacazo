## Instalacion :warning:

### :running: Ejecutar Local

Clonar el proyecto

```sh
- git clone https://github.com/jeroalderete/Jade
```

Ir a la carpeta del proyecto

```bash
  cd Jade
```

Instalar dependencias

```bash
  npm install
```

Start server

```bash
  npm run dev
```
